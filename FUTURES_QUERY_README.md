# 期货查询工具

一个综合的期货价格查询工具，支持多个数据源。

## 功能特点

- **多数据源支持**：
  - 腾讯财经（推荐，稳定且免费）
  - 新浪财经（备用）
  - Yahoo Finance（备用）

- **支持的期货品种**：
  | 代码 | 名称 | 交易所 |
  |------|------|--------|
  | hf_GC | 纽约黄金 | COMEX |
  | hf_SI | 纽约白银 | COMEX |
  | hf_HG | 纽约铜 | COMEX |
  | hf_XAU | 伦敦金（现货黄金） | LBMA |
  | hf_XAG | 伦敦银（现货白银） | LBMA |
  | hf_CL | 纽约原油 | NYMEX |
  | hf_OIL | 布伦特原油 | ICE |
  | hf_NG | 美国天然气 | NYMEX |

- **批量查询**：支持一次查询多个品种
- **自动降级**：当某个数据源不可用时自动切换到下一个

## 使用方法

### 列出所有支持的期货代码

```bash
python query_futures.py --list
```

### 查询所有期货

```bash
python query_futures.py
```

### 查询特定期货

```bash
python query_futures.py hf_GC hf_CL
```

### 指定数据源

```bash
# 使用腾讯财经（推荐）
python query_futures.py --source tencent

# 使用新浪财经
python query_futures.py --source sina

# 使用 Yahoo Finance
python query_futures.py --source yahoo
```

## 文件说明

- `query_futures.py` - 主查询脚本

## 数据源详情

### 腾讯财经接口

- **URL**: `https://qt.gtimg.cn/q=代码1,代码2`
- **优点**: 稳定、快速、支持批量查询
- **格式**: `v_hf_GC="最新价,涨跌幅,开盘价,昨收价,最高价,最低价,时间,买一,卖一,0,1,1,日期,名称";`

### 新浪财经接口

- **URL**: `http://hq.sinajs.cn/list=代码1,代码2`
- **优点**: 作为备用数据源
- **格式**: `var hq_str_hf_GC="名称,开盘价,最高价,最低价,昨收价,最新价,时间";`

### Yahoo Finance

- **优点**: 国际知名数据源
- **缺点**: 可能有访问频率限制
- **需要**: 安装 `yfinance` 库

## 注意事项

1. **腾讯财经和新浪财经使用相同的代码格式**（如 `hf_GC`）
2. 建议优先使用腾讯财经接口，稳定性更好
3. 请合理设置查询频率，避免对数据源造成压力
4. 数据仅供参考，不构成投资建议
