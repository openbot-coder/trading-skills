# 分析场景指南

## 场景一：大盘指数实时监控

### 场景描述
监控A股主要指数的实时行情，快速把握市场整体走势。

### 分析目标
- 了解市场整体强弱
- 观察指数间联动关系
- 判断市场风险偏好

### 分析流程

#### Step 1: 获取主要指数数据
```bash
python get_quote.py sh000001 sh000300 sh000016 sz399001 sz399006
```

#### Step 2: 对比指数表现
```bash
python get_quote.py sh000001 sh000300 sz399001 --detail
```

### 输出示例


- 观察资产间联动
- 评估整体风险暴露

### 分析流程

#### Step 1: 获取各品种实时行情
```bash
# 股票
python crawldata.py quote sh600519 sz000001
```
