# 分析场景指南

## 场景一：个股趋势分析

### 场景描述
对单只股票进行全面的趋势分析，包括价格走势、成交量、技术指标等。

### 分析目标
- 判断当前趋势（上升/下降/震荡）
- 识别支撑位和压力位
- 寻找买卖点信号

### 分析流程

#### Step 1: 获取基础数据
```bash
# 获取日K线数据
npx -y westock-data-clawhub@1.0.4 kline sh600000 --period day --limit 30

# 获取公司简况
npx -y westock-data-clawhub@1.0.4 profile sh600000

# 获取资金流向
npx -y westock-data-clawhub@1.0.4 asfund sh600000
```

#### Step 2: 获取技术指标
```bash
npx -y westock-data-clawhub@1.0.4 technical sh600000 --group all
```

#### Step 3: 数据分析

**价格趋势分析**：
```
1. 计算区间涨跌幅：(最新价 - 30日前价格) / 30日前价格 * 100%
2. 观察均线排列：
   - MA5 > MA10 > MA20 → 多头排列，上升趋势
   - MA5 < MA10 < MA20 → 空头排列，下降趋势
   - 均线缠绕 → 震荡整理
3. 识别支撑位：前期低点、均线位置
4. 识别压力位：前期高点、均线位置
```

**成交量分析**：
```
1. 量价配合：
   - 价涨量增 → 上升趋势得到确认
   - 价涨量缩 → 上升动力不足，警惕回调
   - 价跌量增 → 下降趋势确认
   - 价跌量缩 → 可能见底
2. 异动量能：突放巨量需关注
3. 量比分析：当日成交量/历史平均成交量
```

**技术指标综合判断**：
```
MACD：
- DIF > 0 且 MACD > 0 → 强势
- 金叉信号 → 买入参考
- 死叉信号 → 卖出参考

RSI：
- RSI > 70 → 超买区，谨慎追高
- RSI < 30 → 超卖区，关注反弹
- RSI 背离 → 趋势反转信号

KDJ：
- J值 > 100 → 超买
- J值 < 0 → 超卖
- 金叉/死叉 → 买卖信号
```

### 输出示例

